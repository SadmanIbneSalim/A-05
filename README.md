1️⃣ What is the difference between var, let, and const?
Ans: 	•	var – Old way to declare variables. It’s function-scoped and can be re-declared and updated.
	    •	let – Modern variable. It’s block-scoped, can be updated but cannot be re-declared in the same block.
	    •	const – Constant variable. It’s block-scoped and cannot be updated or re-declared. Must be initialized when declared.

2️⃣ What is the spread operator (...)?
Ans: The spread operator  is used to expand elements of an array or object into individual elements.

3️⃣ What is the difference between map(), filter(), and forEach()?
Ans:	•	forEach() – Loops through an array, does not return anything.
	    •	map() – Loops and returns a new array with transformed values.
	    •	filter() – Loops and returns a new array with elements that match a condition.

4️⃣ What is an arrow function?
Ans: Arrow function is a shorter way to write a function in JavaScript. It uses => instead of the function keyword.

5️⃣ What are template literals?
Ans: Template literals are a way to write strings in JavaScript using backticks ` instead of quotes, which allows variables and expressions inside strings.
